## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Ksanim0/Lab-Web_Edital
```

Entre no diretório do projeto

```bash
  cd Lab-Web_Edital
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
